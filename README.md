Descripción del Proyecto Página destinada a la compra-venta de Libros usados.

El usuario deberá iniciar sesión o registrarse en caso de no contar con usuario o contraseña. En ambas opciones, una vez que la página valida la autenticación del usuario, este será redirigido al inicio de la página web.

Los usuarios pueden realizar las siguientes acciones:

Publicar, visualizar, comprar o vender libros Editar el perfil de Usuario Cambiar la contraseña de Usuario Cerrar Sesión Iniciar sesión en caso de haber cerrado sesión Nota: La opción de editar y eliminar solo lo puede hacer el autor de la publicación del libro.
